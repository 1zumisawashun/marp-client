# marp-client

Marp で作ったプレゼン資料を一元管理しているリポジトリです。

## Overview

- 20240218\_サンプル
- 20240222\_確定申告

## Installation

- clone

```bash
$ git clone git@github.com:1zumisawashun/marp-client.git
$ cd marp-client
```

- install

```bash
$ npm install
```

- ローカル開発用 URL を開き動作確認をする

```bash
$ npm run dev
```

http://localhost:8080/

- 上記の手順で失敗する場合 [Troubleshoot](#Troubleshoot)を確認してください
